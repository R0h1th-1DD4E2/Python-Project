>> To use this python project first head to https://openweathermap.org/api ☁️ 
>> Create an account and generate a API key for the project
>> Now head to the folder where the python 🐍 project is stored and create a ".env" file 
>> Type as given below 
    API_KEY="YOUR_API_KEY_HERE"
>> Run the code ✨✨✨
